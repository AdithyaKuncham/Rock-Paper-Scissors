# Rock-Paper-Scissors (Flutter)

A clean and simple implementation of the classic Rock-Paper-Scissors game, built with Flutter. This app features a modern UI, real-time score tracking, and automatic light/dark mode support.

-----

## ‚ú® Features

* **Classic Game Logic:** Play Rock, Paper, or Scissors against a fair, randomized computer opponent.
* **Real-Time Score Tracking:** A scoreboard keeps track of the User vs. Computer wins for the current session.
* **Instant Feedback:** Immediately see the round's outcome (Win, Lose, or Draw).
* **Responsive UI:** Adapts to different screen sizes and system themes (light/dark mode).
* **Score Reset:** A "Refresh" button in the app bar allows you to reset the score at any time.
